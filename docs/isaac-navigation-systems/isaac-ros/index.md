# Isaac ROS

Isaac ROS provides a collection of hardware-accelerated perception and navigation packages that bridge the gap between NVIDIA's GPU-accelerated algorithms and the Robot Operating System (ROS). These packages enable efficient processing of sensor data for navigation applications.

## Learning Objectives

By completing this chapter, you will:

- Understand the Isaac ROS package ecosystem
- Integrate Isaac ROS packages into navigation workflows
- Leverage GPU acceleration for perception tasks
- Connect Isaac Sim with Isaac ROS for simulation-to-reality transfer

## Topics Covered

- Overview of Isaac ROS packages
- GPU-accelerated perception
- Integration with ROS navigation stack
- Sensor processing pipelines
- Performance optimization

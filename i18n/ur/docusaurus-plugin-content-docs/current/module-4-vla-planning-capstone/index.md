# ماڈیول 4: وژن-لینگویج-ایکشن، منصوبہ بندی اور کیپ سٹون

## ماڈیول کا جائزہ

یہ ماڈیول جدید ترین مصنوعی ذہانت اور روبوٹکس کی تکنیکوں کا احاطہ کرتا ہے۔ آپ سیکھیں گے کہ کیسے روبوٹ دیکھ سکتے ہیں، سمجھ سکتے ہیں، اور عمل کر سکتے ہیں۔

### 📚 **ابواب**

#### باب 1: وژن-لینگویج-ایکشن ماڈلز

یہ باب VLA ماڈلز کی گہرائی میں جاتا ہے:

- **بنیادی ماڈلز**: GPT، CLIP، BERT کو سمجھیں
- **ملٹی موڈل انٹیگریشن**: وژن اور لینگویج کو جوڑیں
- **VLA آرکیٹیکچرز**: RT-1، RT-2، PaLM-E
- **عملی استعمال**: حقیقی روبوٹس میں لاگو کریں
- **چیلنجز**: موجودہ حدود کو سمجھیں

#### باب 2: منصوبہ بندی اور کنٹرول

روبوٹ موشن پلاننگ میں مہارت حاصل کریں:

- **پاتھ پلاننگ**: A*، Dijkstra، RRT الگورتھم
- **ٹریجیکٹری آپٹیمائزیشن**: ہموار موشن بنائیں
- **ٹاسک پلاننگ**: اعلیٰ سطح کی منصوبہ بندی
- **ری ایکٹو پلاننگ**: حرکی رکاوٹیں
- **ہیرف پلاننگ**: گرفت اور چلانا
- **پورے جسم کی منصوبہ بندی**: ہیومنائڈ توازن

#### باب 3: کیپ سٹون پروجیکٹ

اپنا مکمل روبوٹک سسٹم بنائیں:

- **پروجیکٹ کی ضروریات**: کیا بنانا ہے
- **انٹیگریشن گائیڈ**: تمام ماڈیولز کو جوڑیں
- **عملی مراحل**: قدم بہ قدم
- **ٹیسٹنگ**: اپنے سسٹم کی جانچ کریں
- **پریزنٹیشن**: اپنا کام پیش کریں
